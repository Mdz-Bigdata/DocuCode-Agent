# Qwen-Agent 启动指南

## 项目准备已完成！✅

项目环境已经配置好，Qwen-Agent 可以正常运行。

## 快速启动步骤

### 1. 配置 API Key

首先编辑 `.env` 文件，填入您的 DashScope API Key：

```bash
# 打开 .env 文件
# 将 YOUR_DASHSCOPE_API_KEY 替换为您的实际 API Key
```

或者直接在终端中设置环境变量：

```bash
export DASHSCOPE_API_KEY="您的API_Key"
```

### 2. 启动方式选择

#### 方式一：使用简单的命令行示例（推荐先试这个）

```bash
cd /Users/mindezhi/Qwen-Agent
python3 examples/llm_riddles.py
```

#### 方式二：使用 Web GUI 界面

注意：Web GUI 需要 gradio 5.23.1，但当前安装的是 gradio 6.10.0。如果需要完整的 Web GUI，需要降级 gradio：

```bash
pip3 install --user gradio==5.23.1 gradio-client==1.8.0 --break-system-packages
```

然后启动：

```bash
cd /Users/mindezhi/Qwen-Agent
python3 start_gui.py
```

#### 方式三：直接运行项目中的示例

项目提供了很多示例在 `examples/` 目录下：

```bash
cd /Users/mindezhi/Qwen-Agent

# 基础聊天示例
python3 examples/function_calling.py

# Qwen3 示例
python3 examples/assistant_qwen3.py

# 更多示例请查看 examples/ 目录
```

## 获取 API Key

如果您还没有 DashScope API Key，可以：
1. 访问 https://dashscope.console.aliyun.com/
2. 注册/登录阿里云账号
3. 创建 API Key

## 常见问题

### Q: 提示缺少模块怎么办？
A: 运行 `pip3 install --user 模块名 --break-system-packages`

### Q: Python 3.14 兼容性问题？
A: 当前 Python 3.14 太新，部分依赖（如 gradio 5.x）可能不支持。建议使用 Python 3.10-3.13。

### Q: 如何不使用 Web GUI？
A: 可以直接运行命令行示例，它们不需要 gradio。
